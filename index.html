An offline, secure, easy to use password manager

A fully offline, fast and secure password manager for everyone. Easy to use design, advanced security and portability. Written in C# (.NET Core).

Get started Download

Main features

Easy to use

The design is made to be easy to use for all age groups. It has a unified style with a unified color palette. The buttons are intuitive and easy to recognize.

Secure

Thanks to AES-256, SHA 512/256 and inner security measures such as 2 layer encryption, the saved data is almost impossible to crack.

Offline

Fully functional without requiring internet connection. It doesn't need to send/receive any data so your passwords are always secured.

Portable

The generated .safe file is fully portable and really easy to transfer. The password manager itself also can be easily transferred with the single .exe version.

Multi purpose

Along with passwords you can securely save notes, bank cards, wifi passwords and even files inside a safe. All these things are securely encrypted.

Free to use

It's free for everyone, and who knows maybe even Open Source in the future :)

Security and privacy

Multiple options

Combined

Most secure, but not portable

The safes are encrypted with military grade AES-256 bit encryption where the encryption key itself is a SHA hash of the real password. The program also uses double layer encryption which ensures the protection of passwords during runtime.

Store more than just passwords

From notes, wifi passwords, bank cards to files you can store almost everything. All your data is encrypted and secured.

Architecture

The application was entirely written in C#. The design is made using the WPF technology (Windows Presentation Foundation) with XAML markup. The base of the project is the .NET Core 3.1 runtime.

For fast and secure json parsing Json.Net library was used.
